
# esPera

## Descripción

**esPera** es un prototipo sencillo de una aplicación web enfocada en la educación financiera personal. Su objetivo es ayudar a los usuarios a reflexionar antes de realizar gastos, organizar un presupuesto mensual y fomentar el hábito del ahorro.

El proyecto fue desarrollado utilizando únicamente **HTML5** y **CSS3**, con una interfaz responsive diseñada bajo el enfoque **Mobile First**.

---

## Objetivo

Muchas personas conocen cuánto dinero ganan, pero no siempre tienen claridad sobre cuánto gastan ni cómo ciertas decisiones afectan sus metas financieras.

esPera busca responder preguntas como:

- ¿Puedo permitirme esta salida?
- ¿Cuánto estoy destinando al ahorro?
- ¿Estoy respetando mi presupuesto mensual?
- ¿Cómo impactan mis gastos frecuentes en mis finanzas?

---

## Características

### Inicio
- Resumen general de gastos.
- Barra de estado de ánimo para registrar cómo se siente el usuario.
- Diseño limpio y enfocado en la experiencia móvil.

### Presupuesto
- Acceso a la creación de presupuestos.
- Acceso a la consulta del presupuesto actual.
- Frases motivacionales relacionadas con las finanzas personales.

### Crear Presupuesto
- Formulario para registrar categorías de gastos.
- Distribución de gastos por áreas importantes:
  - Alimentación y transporte
  - Vivienda
  - Ahorro
  - Subscripciones y servicios
  - Fondo de emergencia

### Mi Presupuesto
- Visualización simulada de un presupuesto mensual.
- Resumen de categorías y montos asignados.
- Estado financiero general.

### Quiero Salir
- Herramienta de reflexión financiera.
- Permite estimar cuánto costará una salida.
- Considera:
  - Monto estimado.
  - Frecuencia mensual.
  - Tipo de salida.
  - Impacto en el ahorro.
- Genera recomendaciones basadas en los datos ingresados.

### Ahorro
- Visualización de ahorro acumulado.
- Barra de progreso hacia una meta financiera.
- Botones para agregar o retirar ahorro.
- Mensajes motivacionales para fomentar la constancia.

## Estado del Proyecto

Este proyecto corresponde a un **prototipo funcional básico** orientado a demostrar la estructura, navegación y experiencia de usuario de una futura aplicación financiera.

Actualmente utiliza datos simulados y componentes visuales estáticos.

Las siguientes versiones podrían incluir:

- JavaScript para cálculos automáticos.
- Local Storage para guardar información.
- Gráficas dinámicas.
- Sistema de metas financieras.
- Historial de gastos.
- Integración con bases de datos.
